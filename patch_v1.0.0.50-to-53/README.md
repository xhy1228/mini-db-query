# v1.0.0.50 → v1.0.0.53 升级指南

## 本次更新内容

### 1. 版本号显示优化
- 版本号显示在侧边栏"管理平台"下方
- 移除右下角浮动版本号显示

### 2. 启动优化
- 启动服务后直接显示登录页面
- 不再先跳转到数据库配置页面

---

## 升级步骤

### 步骤1：停止当前服务

按 `Ctrl+C` 停止服务

### 步骤2：更新代码文件

将以下文件复制到对应位置：

```
patch_v1.0.0.50-to-53/
├── index.html                    → admin/index.html (覆盖)
└── backend/
    ├── main.py                   → backend/main.py (覆盖)
    ├── version.py                → backend/version.py (覆盖)
    └── api/
        └── stats.py              → backend/api/stats.py (新增)
```

### 步骤3：重启服务

```bash
start.bat
```

### 步骤4：验证

1. 访问 `http://localhost:8000` 应该**直接显示登录页面**
2. 登录后，侧边栏"管理平台"下方显示版本号 `v1.0.0.53`
3. 侧边栏应该有"系统配置"菜单

---

## 注意事项

1. 升级后请按 `Ctrl+F5` 强制刷新浏览器缓存
2. 如果看不到"系统配置"菜单，请确认文件已正确覆盖
3. **必须复制 stats.py 文件到 backend/api/ 目录**
