## v1.0.0 (2026-03-17)

### 🎉 首次发布

### 新增功能

#### 后端服务
- ✅ 用户登录认证（JWT Token）
- ✅ 多角色权限管理（管理员/普通用户）
- ✅ 学校/项目管理
- ✅ 数据库连接配置管理
  - 支持 MySQL 8.0.2+
  - 支持 Oracle（需安装 Instant Client）
  - 支持 SQL Server（需安装 ODBC 驱动）
  - 支持 SQLite（内置测试用）
- ✅ 查询模板管理
- ✅ 智能查询功能
  - 基于模板的条件查询
  - 时间范围筛选
  - SQL预览
- ✅ 查询历史记录
- ✅ 用户权限分配

#### 微信小程序
- ✅ 登录页面
- ✅ 首页（学校选择、业务大类）
- ✅ 查询页面（模板选择、条件输入、结果展示）
- ✅ 历史记录页面
- ✅ 个人中心页面

#### 管理平台
- ✅ 用户管理（增删改查）
- ✅ 学校管理
- ✅ 数据库配置管理
- ✅ 查询模板管理
- ✅ 权限分配

#### 部署工具
- ✅ Windows 一键安装脚本（install.bat / install.ps1）
- ✅ 服务启动/停止脚本（start.bat / stop.bat）
- ✅ MySQL 数据库初始化脚本
- ✅ 完整 PDF 文档

### 已知问题

1. Oracle 连接需要安装 Oracle Instant Client
2. SQL Server 连接需要安装 ODBC Driver 17 for SQL Server
3. 小程序需要配置合法域名才能在真机使用

## 版本号说明

版本号格式：`主版本.次版本.修订号`

- **主版本**：重大架构变更或不兼容更新
- **次版本**：新增重要功能模块
- **修订号**：Bug修复、小功能改进

## 更新日志格式

每个版本更新记录包含以下部分：

1. **新增功能** - 新增加的功能
2. **改进** - 对现有功能的优化
3. **修复** - Bug修复记录
4. **已知问题** - 当前版本存在的问题
5. **升级说明** - 从旧版本升级的注意事项
