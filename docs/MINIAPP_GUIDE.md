## 1. 登录小程序

### 首次使用
1. 在微信中搜索"多源数据查询"小程序
2. 打开小程序后，点击"微信一键登录"按钮
3. 授权微信登录，系统将自动获取您的微信信息
4. 如果您是首次使用，需要联系管理员开通权限

### 登录流程图
```
┌─────────────┐
│ 打开小程序   │
└──────┬──────┘
       ▼
┌─────────────┐
│ 点击登录按钮 │
└──────┬──────┘
       ▼
┌─────────────┐
│ 授权微信登录 │
└──────┬──────┘
       ▼
┌─────────────┐
│ 进入首页     │
└─────────────┘
```

### 登录状态
- **已授权**：显示您的姓名和所属学校
- **未授权**：提示"请联系管理员开通权限"
- **已停用**：提示"账号已被停用，请联系管理员"

## 2. 选择学校和业务

### 选择学校
登录后，首页顶部显示您有权限访问的学校列表：
1. 点击学校名称下拉框
2. 选择您要查询的学校
3. 系统自动加载该学校下的数据库配置和查询模板

### 业务大类
选择学校后，页面显示该学校的业务大类：
- 🏫 **教务管理**：学生信息、成绩查询等
- 💰 **财务管理**：收费信息、缴费记录等
- 📚 **图书管理**：借阅记录、图书查询等
- 👥 **人事管理**：教职工信息、考勤记录等
- 🏢 **资产管理**：设备查询、资产盘点等

### 选择业务类型
1. 点击业务大类图标
2. 进入该大类下的具体业务列表
3. 选择要查询的业务类型

## 3. 执行查询

### 智能查询（推荐）
智能查询提供预设的查询模板，无需编写SQL：

1. **选择查询模板**
   - 进入业务类型后，选择查询模板
   - 系统自动加载查询表单

2. **填写查询条件**
   | 字段类型 | 说明 | 示例 |
   |----------|------|------|
   | 文本输入 | 输入关键字 | 学生姓名 |
   | 数字输入 | 输入数字 | 学号 |
   | 日期选择 | 选择日期范围 | 入学时间 |
   | 下拉选择 | 从列表选择 | 班级 |

3. **执行查询**
   - 点击"查询"按钮
   - 等待结果显示

4. **查看结果**
   - 表格展示查询结果
   - 支持排序和筛选
   - 分页浏览大数据

### SQL查询（高级）
适合熟悉SQL的用户：

1. **切换到SQL模式**
   - 在查询页面点击"SQL查询"标签
   - 显示SQL编辑器

2. **编写SQL语句**
   ```sql
   -- 示例：查询学生信息
   SELECT student_id, name, gender, class_name
   FROM students
   WHERE status = '在校'
   AND grade = '2023'
   LIMIT 100;
   ```

3. **安全提示**
   - ⚠️ 仅允许SELECT查询
   - ⚠️ 禁止DROP、DELETE、UPDATE等危险操作
   - ⚠️ 系统会自动过滤敏感关键字

4. **执行查询**
   - 点击"执行"按钮
   - 系统验证SQL安全性
   - 返回查询结果或错误提示

### 查询结果操作
- **排序**：点击列标题排序
- **筛选**：输入关键词筛选结果
- **分页**：翻页浏览更多数据
- **详情**：点击行查看详细信息

## 4. 查看历史记录

### 访问历史记录
1. 点击底部导航"历史"标签
2. 显示最近的查询记录列表

### 历史记录内容
每条记录包含：
- 📅 **查询时间**
- 🏫 **所属学校**
- 📋 **查询名称**
- 📊 **结果条数**
- ⏱️ **执行耗时**
- ✅ **执行状态**

### 历史记录操作
- **重新执行**：点击记录可重新执行相同查询
- **查看详情**：查看执行的SQL和参数
- **删除记录**：左滑删除单条记录

## 5. 数据导出

### 导出流程
1. 执行查询后，在结果页面点击"导出"按钮
2. 选择导出格式：
   - 📊 **Excel** (.xlsx)
   - 📄 **CSV** (.csv)
3. 确认导出范围：
   - 当前页
   - 全部数据（有上限）
4. 等待导出完成
5. 选择分享方式：
   - 发送到邮箱
   - 保存到微信收藏
   - 分享给好友

### 导出限制
- 单次最多导出 **10,000** 条数据
- 导出文件有效期 **24小时**
- 每日最多导出 **50** 次

## 6. 常见问题

### Q1: 登录后提示"未授权"怎么办？
**A**: 请联系管理员开通权限，需要提供：
- 您的姓名
- 手机号
- 所属学校/部门

### Q2: 查询结果显示"连接超时"？
**A**: 可能原因：
1. 查询数据量过大，请添加更多筛选条件
2. 网络不稳定，请稍后重试
3. 数据库服务器繁忙，请错峰查询

### Q3: 如何修改个人信息？
**A**: 
1. 点击"我的"标签
2. 点击"个人信息"
3. 修改可编辑的字段
4. 点击"保存"

### Q4: 忘记密码怎么办？
**A**: 小程序使用微信登录，无需密码。如遇登录问题：
1. 检查微信是否已更新
2. 清除小程序缓存后重试
3. 联系管理员重置账号

### Q5: SQL查询报错怎么办？
**A**: 常见错误：
- **语法错误**：检查SQL语法
- **表不存在**：确认表名是否正确
- **权限不足**：联系管理员确认查询权限
- **超时**：优化SQL，减少查询数据量

### Q6: 导出文件找不到？
**A**: 
1. 检查微信收藏
2. 检查邮箱垃圾邮件
3. 重新执行导出操作

### Q7: 如何反馈问题？
**A**: 
1. 在"我的"页面点击"意见反馈"
2. 描述问题详情
3. 可上传截图
4. 提交后等待管理员回复

## 操作技巧

### 快速查询技巧
1. **收藏常用查询**
   - 执行查询后点击"收藏"
   - 下次从收藏夹快速访问

2. **使用历史记录**
   - 相同查询无需重复填写
   - 从历史记录快速重新执行

3. **条件组合**
   - 多个条件同时使用可精确定位
   - 减少返回数据量，加快查询速度

### 性能优化建议
- 添加时间范围限制
- 使用精确查询条件
- 避免使用 `SELECT *`
- 合理使用 `LIMIT`

1. **账号安全**
   - 不要将账号借给他人使用

2. **数据安全**
   - 导出数据仅供工作使用

   - 定期清理导出文件

3. **操作规范**
   - 只查询工作相关数据
   - 禁止尝试越权操作
   - 发现漏洞及时上报
