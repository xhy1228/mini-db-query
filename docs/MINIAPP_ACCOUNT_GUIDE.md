## 一、小程序账号申请

### 1.1 访问注册页面

打开浏览器，访问微信公众平台：
**https://mp.weixin.qq.com**

点击右上角【立即注册】按钮

### 1.2 选择账号类型

在注册页面选择【小程序】

```
┌─────────────────────────────────────┐
│                                     │
│    订阅号    服务号    小程序    企业微信    │
│                                     │
│    [点击选择"小程序"]                    │
│                                     │
└─────────────────────────────────────┘
```

### 1.3 填写账号信息

**第一步：填写邮箱和密码**

- 邮箱：使用未注册过微信公众平台的邮箱
- 密码：设置登录密码
- 验证码：填写图片验证码

点击【注册】按钮

**第二步：邮箱验证**

- 登录邮箱查收验证邮件
- 点击邮件中的激活链接

**第三步：信息登记**

选择主体类型：
- 个人：适合个人开发者（功能有限制）
- 企业：适合公司使用（功能完整）

填写信息：
- 企业名称
- 营业执照注册号
- 管理员姓名、身份证、手机号
- 管理员微信扫码验证

**第四步：小程序信息**

- 小程序名称：如"多源数据查询"
- 小程序简介：如"多源数据库智能查询工具"
- 小程序头像：上传logo图片

## 二、小程序基本信息配置

### 2.1 登录小程序后台

访问：https://mp.weixin.qq.com
使用管理员微信扫码登录

### 2.2 配置小程序信息

进入【设置】→【基本设置】

**必填信息：**

| 项目 | 说明 |
|------|------|
| 小程序名称 | 2-20个字符，建议包含核心功能关键词 |
| 小程序简介 | 4-120个字，描述功能特点 |
| 小程序头像 | 108×108像素以上，PNG/JPG格式 |
| 服务类目 | 选择"工具-信息查询" |

**服务类目选择：**

```
设置 → 基本设置 → 服务类目

添加类目：
  └── 工具
       └── 信息查询
            └── 确定
```

## 三、服务器域名配置（重要）

### 3.1 进入域名配置

路径：【开发管理】→【开发设置】→【服务器域名】

### 3.2 配置域名

点击【修改】，添加以下域名：

**request合法域名：**
```
https://your-domain.com
```

**uploadFile合法域名：**
```
https://your-domain.com
```

**downloadFile合法域名：**
```
https://your-domain.com
```

### 3.3 域名配置要求

| 要求 | 说明 |
|------|------|
| 协议 | 必须是 HTTPS |
| 域名 | 必须已备案 |
| 证书 | 必须有效 |
| 格式 | 不含端口号和路径 |

**配置示例：**

```
┌────────────────────────────────────────┐
│ 服务器域名配置                            │
├────────────────────────────────────────┤

│ ┌────────────────────────────────────┐ │
│ │ https://api.example.com            │ │
│ └────────────────────────────────────┘ │
│                                        │
│ uploadFile合法域名                       │
│ ┌────────────────────────────────────┐ │
│ │ https://api.example.com            │ │
│ └────────────────────────────────────┘ │
```

## 四、获取关键参数

### 4.1 获取AppID和AppSecret

路径：【开发管理】→【开发设置】

**AppID（小程序ID）：**
- 位置：开发设置页面顶部
- 格式：wx开头，如 wx1234567890abcdef
- 用途：小程序配置文件

**AppSecret（小程序密钥）：**
- 位置：开发设置 → AppSecret
- 操作：点击【生成】或【重置】
- 注意：生成后只显示一次，请立即复制保存

```
┌────────────────────────────────────────┐
│ 开发设置                                 │
├────────────────────────────────────────┤
│                                        │
│ AppID(小程序ID)                          │
│ ┌────────────────────────────────────┐ │
│ │ wx1234567890abcdef                 │ │
│ └────────────────────────────────────┘ │
│                                        │
│ AppSecret(小程序密钥)                     │
│ ┌────────────────────────────────────┐ │
│ │ ********************************   │ │
│ └────────────────────────────────────┘ │
│ [生成] [重置]                            │
│                                        │
└────────────────────────────────────────┘
```

### 4.2 参数配置位置

**小程序端配置：**

文件：`miniapp/project.config.json`
```json
{
  "appid": "wx1234567890abcdef"
}
```

**后端配置：**

文件：`backend/.env` 或环境变量
```
WECHAT_APPID=wx1234567890abcdef
WECHAT_SECRET=你的AppSecret
```

## 五、开发者工具使用

### 5.1 下载开发者工具

访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

选择对应系统版本下载安装

### 5.2 登录开发者工具

- 打开开发者工具
- 微信扫码登录
- 选择【小程序项目】

## 七、常见问题

### Q1：域名未备案怎么办？
- 个人域名需要完成ICP备案
- 可使用云服务商提供的备案服务

### Q2：AppSecret忘记了怎么办？
- 登录小程序后台
- 开发设置 → AppSecret → 重置
- 重置后旧的Secret立即失效

### Q3：开发者工具无法预览？
- 检查AppID是否正确
- 检查网络连接
- 尝试清除缓存

### Q4：真机无法请求服务器？
- 检查服务器域名是否已配置
- 确认域名使用HTTPS
- 检查SSL证书是否有效
