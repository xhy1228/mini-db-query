# 升级包: v1.0.2.08 → v1.0.2.09

## 快速升级

### Windows用户
1. 将此目录解压到项目根目录
2. 双击运行 `upgrade.bat`
3. 重启服务

### 手动升级
1. 备份现有数据
2. 将 `backend/` 和 `miniapp/` 目录复制到项目目录覆盖
3. 重启服务

## 主要更新

### 🐛 Bug修复

1. **管理平台添加用户保存失败** - 修复路由不匹配问题
2. **数据库配置保存无反应** - 修复前端API调用路径错误

### 📝 技术细节

- 前端用户管理API从 `/manage/users` 改为 `/admin/users`
- 前端数据库配置API路径修正
- 与后端路由定义保持一致

## 无需数据库升级

此版本为Bug修复，无需执行数据库脚本。
