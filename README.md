## 技术栈

### 后端服务
- **语言**: Python 3.10+
- **Web框架**: FastAPI (高性能异步框架)
- **数据库连接**: SQLAlchemy + oracledb/pymysql/pyodbc
- **缓存**: Redis (可选)
- **任务队列**: Celery (可选，用于异步导出)

### 微信小程序
- **开发框架**: 微信小程序原生 / uni-app
- **UI组件**: Vant Weapp / uView

### 管理平台（后期）
- **前端**: Vue 3 + Element Plus
- **后端**: 复用API服务

## 部署架构

