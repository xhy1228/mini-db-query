# 升级包: v1.0.2.09 → v1.0.2.11

## 快速升级

### Windows用户
1. 将此目录解压到项目根目录
2. 双击运行 `upgrade.bat`
3. 重启后端服务

### 手动升级
1. 备份现有数据
2. 将 `backend/` 和 `miniapp/` 目录复制到项目目录覆盖
3. 重启后端服务
