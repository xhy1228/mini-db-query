📖 使用帮助

步骤1：学校管理
先添加学校

步骤2：数据库配置
选择学校，配置 MySQL/Oracle/SQLServer 连接

步骤3：业务大类
创建业务分类，如"消费业务"、"门禁业务"

步骤4：查询模板
创建查询模板，编写 SQL 语句

步骤5：功能绑定
把模板绑定到学校，指定使用哪个数据库

基本格式：

SELECT 字段1 AS 中文名, 字段2 AS 中文名 FROM 表名 WHERE 1=1

带条件：

SELECT * FROM 表名 WHERE 字段 = '值'

示例 - 消费查询：

SELECT CARDID as 卡号, TRANAMT as 金额, TRATIME as 时间 
FROM DATA_CARD_CONSUME 
WHERE 1=1 
<if card_id> AND CARDID = '<card_id>' </if>

字段配置用于用户查询时输入条件值

{
  "id": "card_id",
  "label": "卡号",
  "column": "CARDID",
  "type": "text",
  "operator": "="
}

参数说明：

  • id: 字段唯一标识
  • label: 显示名称
  • column: 对应数据库字段
  • type: 类型 (text/number/date)
  • operator: 操作符 (=/LIKE/>=/<=)

v1.2.0 新功能：支持一个学校配置多个数据库

在"功能绑定"页面：

  • 选择学校
  • 选择查询模板
  • 选择该模板使用哪个数据库

好处：不同查询可以用不同的数据库

Q: 为什么查询失败？
A: 检查SQL模板语法，确认数据库连接正常

Q: 小程序看不到功能？
A: 确保已在"功能绑定"页面绑定模板到学校

Q: 管理员无法使用SQL查询？
A: 检查用户权限是否绑定了学校
